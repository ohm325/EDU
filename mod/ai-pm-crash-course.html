AI Product Manager

Crash Course ฉบับเร่งรัด สำหรับ IT Manager

ผู้สอน: Gemini ผู้เรียน: คุณโอม
Module 1: ปูพื้นฐาน AI ในโลกธุรกิจ
เป้าหมาย: แยกแยะได้ว่า AI แบบไหนเหมาะกับงานอะไร และอะไรคือข้อจำกัด
  • ความต่างของ AI, Machine Learning และ Deep Learning
  • รู้จัก Supervised, Unsupervised และ RL (ฉบับเข้าใจง่าย)
  • เจาะลึก Generative AI และ LLM
  • ด้านมืดของ AI: Bias, Hallucination, และจริยธรรม
Mini-Project
เลือกปัญหาในแผนก IT ของโอม 1 อย่าง แล้ววิเคราะห์ว่าควรใช้ AI ประเภทไหนมาแก้ปัญหา
Module 2: วงจรชีวิตโปรเจกต์ AI
เป้าหมาย: บริหารจัดการโปรเจกต์ AI ให้รอดตั้งแต่ต้นจนจบ (เชื่อมโยงวิชา 96407)
  • AI Lifecycle vs Traditional SDLC (ความต่าง)
  • การเขียน AI PRD สิ่งที่ต้องมีในเอกสารสเปก
  • เข้าใจคอนเซปต์ MLOps เบื้องต้น
Mini-Project
เขียน Draft PRD สั้นๆ สำหรับฟีเจอร์ "AI Chatbot ช่วยตอบคำถามพนักงานใหม่ (HR)"
Module 3: วัดผลความฉลาด AI
เป้าหมาย: คุยกับ Data Scientist รู้เรื่อง และดูออกว่าโมเดลดีพอจะใช้งานจริงหรือยัง
  • พื้นฐาน Data Pipeline (ETL) ทำไมข้อมูลขยะ ถึงได้ AI ขยะ
  • ตัวชี้วัด Machine Learning (Accuracy, Precision, Recall, F1)
  • ตัวชี้วัด LLM (Cost per Token, Latency, ROUGE Score)
Mini-Project
วิเคราะห์ Case Study โมเดลตรวจจับสแปมที่มี Accuracy 99% แต่กลับใช้งานจริงไม่ได้
Module 4: Python สำหรับ AI PM
เป้าหมาย: เขียนโค้ดเป็นอาวุธ เพื่อทดสอบไอเดีย (PoC) ได้ด้วยตัวเอง (เชื่อมโยงวิชา 96414)
  • การเรียกใช้ API (ทำความเข้าใจ JSON และ REST API)
  • ทบทวนโครงสร้าง Python ที่จำเป็น (Variables, Dictionaries)
  • การต่อ API ของ OpenAI เพื่อสร้าง Text Summarizer
Mini-Project
เขียน/จำลองโค้ด Python 15 บรรทัด เพื่อส่งข้อความไปให้ AI ช่วยแปลภาษา
Module 5: ศิลปะการขายไอเดีย
เป้าหมาย: แปลงตัวเลขเทคนิค ให้เป็น Business Value เพื่อให้ผู้บริหารอนุมัติงบ
  • การตั้งสมมติฐานทางธุรกิจ (Business Hypothesis) สำหรับ AI
  • AI Value Translation (แปลงความแม่นยำ เป็นการลดต้นทุน)
  • การทำ Data Storytelling เบื้องต้น
Capstone Project
จำลองการ Pitching โปรเจกต์ AI ให้ CEO ฟังใน 3 ย่อหน้า (เน้นความคุ้มค่าและลดความเสี่ยง)
